# WhatsApp Message Sender Bot

- Lê números de `numeros.txt` e mensagem de `mensagem.txt`  
- Formata números para padrão brasileiro (+55)  
- Envia mensagens pelo WhatsApp com Venom-bot  
- Delay aleatório entre 30s e 2min entre envios  
- Registra erros em `erros.txt`  
- Para usar: `npm install venom-bot` → `node seu_script.js`  
- Na primeira execução, escaneie o QR Code para autenticar  
- Use com responsabilidade e respeite a privacidade
